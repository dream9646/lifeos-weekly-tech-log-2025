============================================================
指標 × Linux Debug（BMC 工程師必備 6+1 項）
============================================================

1) 字串指標（char *）
------------------------------------------------------------
用途:
  - sysfs path
  - shell path
  - JSON key
  - mapping 表格的字串欄位

例:
  char *path = "/sys/class/gpio/export";

------------------------------------------------------------

2) buffer 指標（uint8_t *）
------------------------------------------------------------
用途:
  - I2C / SMBus / PMBus
  - sensor block read
  - CPLD register dump

例:
  uint8_t buf[32];

------------------------------------------------------------

3) struct 指標（struct foo *）
------------------------------------------------------------
用途:
  - LED table
  - watchdog config
  - driver parameter table

例:
  struct LedInfo *info;

------------------------------------------------------------

4) 2D 資料（pointer + length）
------------------------------------------------------------
用途:
  - PMBus block (最多 32 bytes)
  - SMBus data with length

例:
  read_block(addr, buf, &len);

------------------------------------------------------------

5) out-pointer（int *value_out）
------------------------------------------------------------
用途:
  - 傳回多個值
  - SMBus read + 返回狀態碼

------------------------------------------------------------

6) 函式指標（加分）
------------------------------------------------------------
用途:
  - callback
  - daemon event handler

------------------------------------------------------------

7) strace（Linux Debug 必備全集）
------------------------------------------------------------
用途:
  - 看 open / read / write 是否成功
  - 看 ioctl 編號
  - 看錯誤碼（如 EINVAL, ENOENT）

指令:
  strace -f -o log.txt ./your_app

看點:
  - 哪裡 return -1？
  - errno = ?
  - 實際 open 到的路徑是否存在？
