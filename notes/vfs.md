什麼是 VFS？（BMC 工程師最小理解）
============================================================

關鍵字:定義
內容:
  VFS = Linux 內核中的「虛擬檔案系統層」
  作用：把所有不同類型的裝置都抽象成 “檔案”

例：
  - GPIO → /sys/class/gpio/...
  - LED → /sys/class/leds/...
  - I2C  → /dev/i2c-0
  - WDT  → /dev/watchdog
  - Network → /proc/net/...

核心概念：
  你用 open/read/write 時，
  並不是直接操作硬體，
  而是 VFS 把操作轉給對應的 driver。

一句話：
  「VFS 讓你可以用檔案的方式控制硬體。」
